# 3D Models

Place your 3D model files here.

## Expected models:
- player/ - Player character model and animations
- enemies/grunt/ - Grunt enemy model and animations
- enemies/heavy/ - Heavy enemy model and animations
- weapons/ - Weapon models
- environment/ - Environment props

## Supported formats:
- .glb (recommended)
- .gltf
- .fbx (may require import settings)
